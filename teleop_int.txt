Step 1: copy and paste manual_pilot.cpp into ros2_ws/src/controller_pkg/src

Step 2: Build the controller library again:
colcon build --packages-select controller_pkg
source install/setup.bash

Step 3: run the following terminals with following commands

Terminal 1: ros2 launch simulation simulation.launch.py (Starts Unity).

Terminal 2: Run that given controller activation command (Takes off/Hovers).

Terminal 3: ros2 run controller_pkg manual_pilot (Starts the script above).

Terminal 4: ros2 run teleop_twist_keyboard teleop_twist_keyboard (The keyboard controls, control the drone in this terminal).
